# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.9** 입니다.

## 이번 업데이트(v1.9) - NPC/조개 동선 안정화 + 3D 상호작용 보강

이번 업데이트는 실제 플레이에서 체감되는 문제를 기준으로 **20개 개선 묶음**을 반영했습니다.

1. 3D 카메라 기준 이동 벡터 재검증/유지
2. 플레이어 3D 모델의 facing 회전 동기화
3. NPC 3D 모델의 facing 회전 동기화
4. NPC 이동 벡터(`vx/vy`) 추적 강화
5. 비마루 NPC의 물 진입 방지
6. 비마루 NPC가 물에 빠졌을 때 자동 복귀
7. NPC 초기 스폰 시 물 타일 안전 보정
8. shell 스폰 위치를 물 내부에서 해안가 육지 인접 타일로 변경
9. shell 픽업 반경 상향(체감 수집성 개선)
10. 플레이어 깊은 물 진입 시 마지막 안전 좌표로 복귀
11. 마지막 안전 좌표(`lastSafeX/Y`) 상태 추적 추가
12. 3D 대화 전용 오버레이 UI 추가
13. 3D 대화 UI에 선택 버튼(1/2/종료) 추가
14. 대화 오버레이 클릭으로 선택/종료 가능
15. 상호작용 프롬프트 자동 표시(`E: 집/상점/박물관/대화`)
16. 메시지 영역을 프롬프트 우선 표시로 개선
17. 대화 중 3D UI 가시성 안정화
18. 2D 대화/3D 대화 공존 구조 정리
19. 버전/상태 표시 동기화(`v1.9`)
20. 관련 문서/조작 설명 갱신

## 조작

- 이동: `WASD` / 방향키 (3D 모드에서는 카메라 시점 기준)
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc` (3D에서는 대화창 버튼 클릭도 가능)
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
