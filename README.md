# 힐링 포레스트: 아일랜드 라이프 (2D)

요청한 대로, 질문 없이 순차 고도화를 진행하고 있습니다.
이번 커밋은 **전반 고도화 2단계**입니다.

## 이번 단계(2단계) 핵심

- **주민 관계도 시스템**
  - NPC별 호감도 수치 표시
  - 대화/선물/퀘스트 제출로 관계도 상승
- **일일 이벤트 + 달력 강화**
  - 날짜/계절/날씨에 더해 `오늘의 이벤트` 적용
  - 이벤트에 따른 보너스(코인/꽃 등) 반영
- **상점 로테이션(일일 변동)**
  - 씨앗/가구/물고기 판매가가 날짜 기반으로 변동
  - 상점이 매일 조금씩 달라지는 경제 루프 추가
- **인테리어 편집 고도화**
  - 단순 랜덤 배치 → 가구 선택/이동/회전 지원
  - `I/J/K/L` 이동, `T` 회전, `Tab` 선택 전환
- **기존 핵심 루프 유지**
  - 다리 건설 + 다리 위 낚시
  - 타이밍 낚시 미니게임
  - 집 구매/업그레이드/입장
  - 농사(심기/성장/수확)
  - 저장/불러오기

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 대화 선택: `1` / `2`, 종료: `Esc`

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
