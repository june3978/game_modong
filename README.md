# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.1** 입니다.

## 이번 업데이트(v1.1) - 텍스처/모델 밀도 확장

- **실사형 텍스처 소스 확장**
  - 지면/흙/목재/지붕에 외부 PBR 스타일 텍스처를 적용해 월드 디테일을 강화
  - 텍스처 로드 실패 시 자동 폴백 텍스처로 렌더가 깨지지 않도록 구성
- **Free 3D 모델 월드 배치**
  - GLTFLoader를 추가해 무료 공개 glTF 샘플 모델(소품)을 월드에 로드
  - 모델에 그림자/수광 처리를 적용해 조명 환경과 자연스럽게 결합
- **기존 실시간 환경 렌더 유지**
  - Three.js 기반 조명/그림자/안개/톤매핑을 유지한 채 콘텐츠 밀도 업그레이드

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
