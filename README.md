# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v0.7** 입니다.

## 이번 업데이트(v0.7) - 풀 3D 월드 확장

- **풀 3D 월드 렌더러 도입 (캔버스 기반 소프트웨어 3D)**
  - 상단 `3D뷰` 버튼으로 기존 2D와 실시간 전환
  - 월드 타일/플레이어/NPC/자원/집/다리를 3D 오브젝트로 렌더링
  - 시간 흐름에 따라 하늘 톤이 변하는 3D 라이트 연출 적용
- **집 내부 캐릭터 가시성 안정화 유지**
  - 실내에서 캐릭터가 항상 보이도록 렌더 경로 유지
- **3D 카메라 조작 추가**
  - `Q/C`: 카메라 회전
  - `Z/X`: 카메라 줌 인/아웃
- **렌더 표면 자동 전환**
  - 실외 3D 모드에서는 전용 3D 캔버스 뷰 사용
  - 실내 진입 시 자동으로 2D 실내 뷰로 안전 전환

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
