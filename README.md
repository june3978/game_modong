# 숲속 쉼터 - 2D 힐링 게임

`숲속 쉼터`는 브라우저에서 바로 실행할 수 있는 작은 2D 힐링 게임입니다.  
별도 게임 엔진이나 복잡한 프로그램 없이, `index.html` 파일 하나에 화면/디자인/게임 동작이 모두 들어 있습니다.

> 프로그래밍을 잘 몰라도 괜찮습니다. 아래 설명대로 파일을 열거나 GitHub Pages를 켜면 실행할 수 있습니다.

---

## 지금 이 저장소에 들어있는 것

| 파일 | 설명 |
| --- | --- |
| `index.html` | 실제 게임 파일입니다. 이 파일을 브라우저로 열면 게임이 실행됩니다. |
| `README.md` | 지금 보고 있는 안내 문서입니다. |
| 이미지 파일들 | 기존 저장소에 있던 이미지 자료입니다. 현재 게임은 주로 캔버스 도형으로 그려집니다. |

---

## 게임 특징

- 2D 캔버스 기반 숲속 산책 게임
- WASD/방향키 이동
- `E` 키로 나무, 꽃, 연못, 모닥불, NPC와 상호작용
- 목재/꽃잎/물고기/별조각 수집
- 오늘의 퀘스트와 보상
- 힐링 티 세트 제작
- NPC `토토`와 대화
- 자동 저장/수동 저장/초기화 기능

## 설치 없이 가장 쉽게 실행하는 방법

### 방법 1. GitHub에서 `index.html`만 다운로드해서 실행하기

프로그램을 아무것도 설치하지 않은 상태라면 이 방법이 가장 쉽습니다.

1. GitHub 저장소 화면에서 `index.html` 파일을 클릭합니다.
2. 오른쪽 위의 `Raw` 버튼을 누릅니다.
3. 열린 화면에서 `Ctrl + S`를 누릅니다.
4. 파일 이름이 `index.html`인지 확인하고 저장합니다.
5. 저장한 `index.html` 파일을 더블클릭합니다.
6. Chrome 브라우저에서 게임이 열립니다.

> 주의: GitHub에서 보이는 코드 화면은 “코드 보기” 화면일 뿐이라 게임이 실행되지 않습니다. 반드시 `Raw`로 저장하거나 GitHub Pages를 사용해야 합니다.

## 링크처럼 실행하고 싶다면: GitHub Pages 사용하기

GitHub Pages를 켜면 다른 사람에게 링크를 보내서 게임을 보여줄 수 있습니다.

1. GitHub 저장소에서 `Settings` 탭을 클릭합니다.
2. 왼쪽 메뉴에서 `Pages`를 클릭합니다.
3. `Build and deployment` 영역을 찾습니다.
4. `Source`를 `Deploy from a branch`로 선택합니다.
5. `Branch`를 `main`으로 선택합니다.
6. 폴더는 `/ (root)`로 선택합니다.
7. `Save`를 누릅니다.
8. 1~2분 정도 기다립니다.
9. 화면에 생성된 주소로 접속합니다.

주소는 보통 아래처럼 생겼습니다.

```text
https://사용자이름.github.io/저장소이름/
```

## 터미널을 사용할 수 있는 경우 실행 방법

Python이 설치되어 있다면 아래 명령으로 로컬 서버를 열 수 있습니다.

```bash
python3 -m http.server 4173 --bind 127.0.0.1 --directory .
```

그 다음 Chrome 주소창에 아래 주소를 입력합니다.

```text
http://127.0.0.1:4173/index.html
```

서버를 끄고 싶으면 터미널에서 `Ctrl + C`를 누르면 됩니다.

## 코드 수정은 어디에서 하나요?

모든 게임 코드는 `index.html` 안에 있습니다.

### 1. 화면 글자나 버튼을 바꾸고 싶을 때

`index.html`의 `<body>` 영역을 수정합니다.  
예를 들어 제목, 설명, 버튼 문구, 오른쪽 패널 내용이 여기에 있습니다.

### 2. 색깔이나 크기, 배치를 바꾸고 싶을 때

`index.html` 위쪽의 `<style>` 영역을 수정합니다.

### 3. 게임 규칙을 바꾸고 싶을 때

`index.html` 아래쪽의 `<script>` 영역을 수정합니다.  
이곳에 이동, 상호작용, 점수, 저장, 퀘스트 로직이 들어 있습니다.

## 자주 하는 질문

### Q. GitHub 화면에서 게임이 바로 실행되지 않아요.

정상입니다. GitHub의 파일 화면은 코드를 보여주는 화면입니다.  
실행하려면 `index.html`을 다운로드해서 열거나 GitHub Pages를 켜야 합니다.

### Q. PowerShell에 명령어를 쳤는데 안 돼요.

Python이 설치되어 있지 않으면 로컬 서버 명령이 동작하지 않을 수 있습니다.  
그럴 때는 터미널을 쓰지 말고 `index.html`을 다운로드해서 더블클릭하거나 GitHub Pages를 사용하세요.

### Q. 스마트폰에서도 되나요?

현재는 키보드 조작 중심이라 PC/노트북 Chrome에서 플레이하는 것이 가장 좋습니다.  
추후 모바일 터치 버튼을 추가하면 스마트폰에서도 더 편하게 플레이할 수 있습니다.
