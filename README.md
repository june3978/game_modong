# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.2** 입니다.

## 이번 업데이트(v1.2) - 더 현실적인 환경 렌더링

- **PBR 텍스처 고도화**
  - 잔디/흙/목재/지붕에 알베도 + 노멀 + 러프니스 맵을 적용해 표면 디테일을 강화
  - 외부 텍스처 로드 실패 시 폴백 텍스처를 사용해 안정적으로 동작
- **조명/노출/안개 리얼리즘 강화**
  - 물리 기반 라이트 설정(physicallyCorrectLights)
  - 날씨/시간대 기반으로 태양광 강도, 환경광, 톤매핑 노출, 안개 거리 동적 조절
- **환경 이펙트 추가**
  - 하늘 돔(sky dome) 추가
  - 비 오는 날 파티클 레인 렌더링으로 분위기 강화
- **월드 밀도 유지 + Free 모델 활용**
  - 기존 free glTF 소품 배치를 유지해 월드 볼륨감 지속 강화

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
