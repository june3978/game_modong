# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.5** 입니다.

## 이번 업데이트(v1.5) - 시점기반 조작 + 월드 건물 상호작용 + 로딩 신뢰성 개선

- **시점 기준 이동 조작 추가**
  - 3D 모드에서 `WASD`/방향키가 카메라 시점(yaw) 기준으로 이동하도록 변경
  - 카메라를 돌리면 조작 방향도 같이 회전해 자연스러운 3D 컨트롤 제공
- **상점/박물관 월드 건물 구현 강화**
  - 월드에 상점/박물관 건물을 3D 메시로 추가
  - 건물 입구 근처에서 `E` 상호작용으로 상점/박물관 UI를 열도록 연결
- **텍스처 로딩 개선/검증 유지**
  - 기존 일부 차단되던 텍스처 소스를 공개 접근 가능한 텍스처 소스로 교체
  - 텍스처/모델 성공·실패 수치는 HUD, 마을보드, `window.__renderStats`에서 확인 가능
- **캐릭터 애니메이션 유지**
  - 플레이어/NPC 리깅 캐릭터의 걷기/대기 애니메이션 유지

## 안내 (자산 소스 관련)

- 요청하신 TurboSquid/Quixel 직접 스크랩 방식은 라이선스/접근 정책 이슈가 있어 코드에 하드코딩하지 않았습니다.
- 대신 현재는 공개 접근 가능한 텍스처 소스와 내부 폴백 메시를 사용해 안정성을 우선 확보했습니다.

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
