# 힐링 포레스트: 시뮬레이션 에디션 (2D)

모동숲 감성에서 영감을 받은 브라우저 기반 2D 힐링 게임입니다.
이번 버전은 단순 데모를 넘어 **NPC AI 시뮬레이션 + 실제 대화 시스템** 중심으로 고도화했습니다.

## 주요 특징

- 대형 타일 월드 + 카메라 추적
- 날씨/시간대 변화 (맑음·비 / 아침·노을·밤)
- 자원 채집 + 낚시 + 인벤토리/레벨/코인 시스템
- **NPC AI (유틸리티 기반 의사결정)**
  - 행동 상태: 휴식, 산책, 채집, 낚시, 대화
  - 기분/에너지 기반 상태 전이
  - NPC끼리 사회적 상호작용(대화), 자동 행동 피드 표시
- NPC와 실제 대화 UI(선물/재료전달/잡담) 및 친밀도 루프
- 퀘스트(마을 잔치 준비)와 보상 루프

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E` (대화 창 오픈)
- 낚시: `Space`
- 대화 선택: `1` / `2`, 닫기: `Esc`
- 휴식: 캠프 근처에서 `캠프에서 휴식` 버튼

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
