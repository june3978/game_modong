# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.6** 입니다.

## 이번 업데이트(v1.6) - PolyHaven 우선 텍스처 체인 + PBR 머티리얼 고도화

- **PolyHaven 우선 텍스처 로딩 체인 적용**
  - 텍스처 URL을 단일 경로에서 다중 후보 체인으로 변경
  - 1순위로 PolyHaven 텍스처를 시도하고, 실패 시 백업 공개 텍스처로 자동 폴백
- **PBR 머티리얼 세부 강화**
  - 잔디/흙/목재/지붕의 albedo/normal/roughness 맵을 유지하면서
  - `normalScale`, `roughness`, `envMapIntensity`를 재조정해 표면 질감과 반사 반응 강화
- **진단 체계 유지**
  - 텍스처/모델 성공·실패 카운트는 HUD, 마을보드, `window.__renderStats`로 확인 가능
- **시점/월드 상호작용 유지**
  - 3D 카메라 기준 이동, 월드 상점/박물관 입구 상호작용 흐름 유지

## 안내 (퀄리티 기대치)

- “오버워치급” 품질은 상용 파이프라인(전용 아트팀, 자체 쉐이더/리깅/애니메이션, 고품질 에셋 제작 공정) 수준이라 현재 단일 파일 웹 프로젝트에서 완전 동일 재현은 현실적으로 어렵습니다.
- 대신 이 프로젝트에서는 **접근 가능한 공개 에셋 + 실시간 PBR/조명 파이프라인 + 검증 가능한 로딩 진단**으로 최대한 가깝게 품질을 끌어올리는 방향으로 계속 고도화하고 있습니다.

## 조작

- 이동: `WASD` / 방향키 (3D 모드에서는 카메라 시점 기준)
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
