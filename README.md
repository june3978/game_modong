# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v0.9** 입니다.

## 이번 업데이트(v0.9) - 그래픽 셀프 피드백 기반 아트 패스

- **3D 월드 아트 패스 강화**
  - 타일 단조로움 개선: 타일별 색 변화(노이즈) + 면 하이라이트/그림자 보강
  - 원근감 개선: 투영 계산에 원근 계수를 강화해 거리 표현 개선
  - 카메라 근처 중심 렌더링으로 밀도 유지 + 화면 퍼포먼스 안정화
- **환경 디테일 대폭 추가**
  - 길/농지/잔디/물 구역의 시각 차별화 강화
  - 농장 울타리, 작물, 덤불 외에 **나무 레이어** 추가
  - 물 반짝임, 길 하이라이트, 최종 비네팅으로 장면 깊이감 향상
- **실내 이동 수정 유지**
  - 집 안에서 플레이어 이동 가능 + 가구 충돌 처리 유지

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
