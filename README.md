# 힐링 포레스트: 디럭스 (2D)

모동숲 감성을 참고해 만든 **브라우저 기반 2D 슬로우 라이프 게임**입니다.
기존 데모에서 한 단계 업그레이드하여 월드 스케일, 카메라 추적, 퀘스트, 인벤토리, 날씨/시간 시스템을 넣었습니다.

## 핵심 특징

- 큰 월드(타일 맵) + 카메라 추적
- 숲/초원/물 지형 기반 자원 채집
- NPC(루나)와 상호작용 + 퀘스트 보상
- 인벤토리/코인/레벨/기분/에너지 시스템
- 낚시 미니 인터랙션
- 날씨(맑음/비) 및 아침-노을-밤 전환

## 조작법

- 이동: `WASD` 또는 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시: `Space`
- 휴식: 캠프 근처에서 `캠프에서 휴식` 버튼

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
