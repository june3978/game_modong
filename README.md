# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.0** 입니다.

## 이번 업데이트(v1.0) - 실제 환경 렌더링 강화

- **Three.js 기반 실제 3D 렌더러 적용**
  - Canvas 의사 3D 렌더를 WebGL 렌더러로 교체
  - Perspective 카메라 + 안개(Fog) + 톤매핑(ACES Filmic) 적용
- **조명/그림자 업그레이드**
  - 태양광(Directional) + 반사광 + Hemisphere Light 구성
  - Soft Shadow(PCF) 활성화로 캐릭터/오브젝트 그림자 표현
  - 시간대 및 날씨에 따라 광량/배경색/안개색 동적 변화
- **실사 느낌 텍스처 강화**
  - CanvasTexture 기반 잔디/흙/나무/지붕 노이즈 텍스처 적용
  - 물은 MeshPhysicalMaterial로 투명도/광택감을 추가
- **월드 밀도 강화 유지**
  - 연못, 긴 다리, 집, 숲, 자원, NPC/플레이어를 전부 3D 메시로 렌더
  - 기존 게임 루프(이동/상호작용/경제/퀘스트)와 연동

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
