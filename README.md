# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v0.6** 입니다.

## 이번 업데이트(v0.6)

- 집 내부에서 플레이어 캐릭터가 보이지 않던 버그 수정
  - 내부 렌더 함수에서 플레이어를 화면 좌표로 직접 그리도록 변경
- 렌더 모드 추가
  - 상단 `3D뷰` 버튼으로 **2D / 3D 스타일 렌더 모드 전환**
  - HUD에 현재 렌더 모드(2D/3D) 표시
- 월드 타일 렌더 개선
  - 2D 기본 렌더와 3D 스타일(입체 음영) 렌더 분리

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
