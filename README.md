# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v0.8** 입니다.

## 이번 업데이트(v0.8) - 풀 3D 아트 스타일 고도화

- **풀 3D 아트 스타일 업그레이드**
  - 타일 음영 수준이 아닌, 카메라 투영 기반 3D 월드 렌더로 전면 개선
  - 잔디/길/농지 구역을 분리 렌더하고, 농장 펜스/작물/덤불/집/다리를 깊이 정렬로 표현
  - 소프트 그림자와 하이라이트를 추가해 입체감 개선
- **카메라 고도화**
  - `Q/C`: 월드 회전
  - `Z/X`: 줌 인/아웃
- **집 내부 이동 버그 수정**
  - 집 안에서도 캐릭터가 실제로 이동하도록 실내 이동 로직 추가
  - 가구 위치와 충돌 처리되어 가구를 관통하지 않음
- **실내/실외 렌더 안정성 유지**
  - 실외는 2D/3D 토글 렌더
  - 실내 진입 시 자동으로 실내 2D 뷰로 전환

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
