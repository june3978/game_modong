# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.7** 입니다.

## 이번 업데이트(v1.7) - 렌더러/건물 품질 패스 강화

- **렌더러 품질 강화**
  - 픽셀 비율 상향(`setPixelRatio`)로 선명도 개선
  - 그림자 타입을 `VSMShadowMap`으로 변경하고 shadow map 해상도를 확대(4096)
  - tone mapping/exposure와 라이트 바이어스를 재튜닝해 입체감 개선
- **건물 모델링 개선**
  - 집/상점/박물관을 단순 박스 형태에서 디테일 건물 스타일로 교체
  - 기초 바닥, 문/손잡이, 창문 유리, 굴뚝, 장식(간판/어닝/엠블럼), 바닥 접지 그림자 추가
- **텍스처 체인 유지**
  - PolyHaven 우선 + 백업 텍스처 체인 로딩은 유지
  - 로딩 성공/실패 수치는 HUD/마을보드/`window.__renderStats`에서 확인 가능

## 안내

- “언리얼급” 최종 퀄리티는 엔진/아트파이프라인 규모 차이로 단번에 동일 재현은 어렵지만,
  현재 구조에서 렌더 설정·머티리얼·모델링 디테일을 단계적으로 올리는 방식으로 최대한 근접하게 고도화 중입니다.

## 조작

- 이동: `WASD` / 방향키 (3D 모드에서는 카메라 시점 기준)
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
