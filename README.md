# 힐링 포레스트: 아일랜드 라이프 (2D)

요청한 대로, 질문 없이 순차 고도화를 진행하고 있습니다.
이번 커밋은 **전반 고도화 5단계(v0.5)** 입니다.

## 이번 단계(5단계) 핵심

- **다리 확장(가시성/이동성 개선)**
  - 연못을 실제로 가로지르는 긴 다리로 조정
  - 이동/낚시 동선이 자연스럽도록 보정
- **박물관 시스템 추가**
  - 물고기/곤충/조개/꽃 기증
  - 기증 보상 및 누적 수집 진행
- **곤충 채집 기능 추가**
  - `G` 키로 곤충 채집 시도
  - 채집 결과를 판매/기증 루프로 연결
- **업적 시스템 추가**
  - 다리 장인 / 큐레이터 / 베스트 프렌드
  - 달성 시 보상 코인 지급
- **주민 일일요청 시스템**
  - 주민별로 매일 다른 재료 요청이 생성
  - 요청 충족 시 코인 보상 + 관계도 상승
- **상태 검증 회귀 강화**
  - 모달 열기/닫기, 상점 액션, HUD 렌더 유지 시나리오 점검

- **모달 닫기 버그 수정**
  - 실행 시 닫히지 않던 창(모달) 표시 충돌 해결
- **주택 평가(인테리어 점수) 추가**
  - 가구 배치/이동/회전에 따라 점수 계산
  - 마을보드에서 점수 확인 가능
  - 5일마다 주택 평가 보너스 코인 지급
- **주민 관계도 시스템**
  - NPC별 호감도 수치 표시
  - 대화/선물/퀘스트 제출로 관계도 상승
- **일일 이벤트 + 달력 강화**
  - 날짜/계절/날씨에 더해 `오늘의 이벤트` 적용
  - 이벤트에 따른 보너스(코인/꽃 등) 반영
- **상점 로테이션(일일 변동)**
  - 씨앗/가구/물고기 판매가가 날짜 기반으로 변동
  - 상점이 매일 조금씩 달라지는 경제 루프 추가
- **인테리어 편집 고도화**
  - 단순 랜덤 배치 → 가구 선택/이동/회전 지원
  - `I/J/K/L` 이동, `T` 회전, `Tab` 선택 전환
- **기존 핵심 루프 유지**
  - 다리 건설 + 다리 위 낚시
  - 타이밍 낚시 미니게임
  - 집 구매/업그레이드/입장
  - 농사(심기/성장/수확)
  - 저장/불러오기

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
