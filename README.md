# 힐링 포레스트: 아일랜드 라이프 (2D)

요청한 피드백을 반영해, 단순 데모에서 **게임다운 루프**를 갖춘 버전으로 개편했습니다.

## 이번 개편 핵심

- **연못 다리 시스템**
  - 다리 키트를 제작/설치하면 연못을 가로질러 이동 가능
  - 다리 위에서도 낚시 가능
- **낚시 타이밍 미니게임**
  - `입질` 타이밍에 훅(Space) → 리일 구간에서 타이밍 바 맞추기
  - 딸깍 1회 낚시가 아닌 단계형 메커니즘
- **말풍선 대화 + 상호작용 연출**
  - NPC 근처 `E`로 대화 시작
  - 대화 중 플레이어/NPC 정지 및 서로 바라보는 상태
  - 말풍선/선택지(1,2)로 잡담/퀘스트 전달/선물
- **UI 전면 개편**
  - 상단 게임 HUD, 우측 인벤토리/목표, 하단 낚시 오버레이
  - 제작/상점/건축 모달 분리
- **수집 재료 활용처 추가**
  - 제작: 가구, 다리 키트, 미끼
  - 퀘스트 제출 및 상점 구매 재화 루프
- **집 구매/업그레이드/입장/인테리어**
  - 집 부지 → 구매(코인) → 업그레이드
  - 문 앞 `E`로 내부 입장
  - 내부에서 `F`로 가구 배치

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 대화 선택: `1` / `2`, 종료: `Esc`

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
