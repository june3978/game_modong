# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.4** 입니다.

## 이번 업데이트(v1.4) - 동작 검증 + 캐릭터 모델링/애니메이션 강화

- **텍스처 로딩 확인 기능 추가**
  - 텍스처 로더의 성공/실패 카운트를 수집
  - HUD 및 마을보드에서 로딩 상태를 수치로 확인 가능
- **캐릭터 모델링 교체**
  - 플레이어/NPC를 단순 캡슐에서 파츠형 3D 리깅 캐릭터로 교체
  - 머리/몸통/팔/다리/가방 구성으로 실루엣 다양성 강화
- **캐릭터 애니메이션 추가**
  - 이동 시 팔/다리 스윙, 대기 시 미세 스웨이 애니메이션 적용
  - NPC도 이동 상태에 따라 자연스럽게 애니메이션 동작
- **모델 로드 폴백 강화**
  - 외부 GLTFLoader가 없거나 로드 실패해도, 로컬 3D 소품이 월드에 자동 생성되어 씬이 비지 않도록 구성
- **검증용 디버그 상태 노출**
  - `window.__renderStats`로 텍스처/모델 로딩 상태를 스크립트에서 점검 가능

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
