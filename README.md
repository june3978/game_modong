# 모동 힐링 포레스트 (2D)

모동숲 감성의 **가벼운 2D 힐링 게임** 데모입니다.
브라우저에서 바로 실행할 수 있습니다.

## 플레이 방법

- 이동: `방향키` 또는 `WASD`
- 낚시: `Space`
- 휴식: 집 근처에서 `집에서 쉬기` 버튼 클릭

## 규칙

- 꽃(🌸), 조개(🐚)를 주우면 점수가 올라갑니다.
- 호수 근처에서 낚시하면 랜덤 보상을 받습니다.
- 움직이면 에너지가 소모되고, 멈추거나 휴식하면 회복됩니다.
- 배경은 아침/노을/밤으로 서서히 변합니다.

## 실행

아래 중 하나로 실행하세요.

```bash
python3 -m http.server 8000
```

실행 후 브라우저에서 `http://localhost:8000` 접속.
