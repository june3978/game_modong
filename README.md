# 힐링 포레스트: 아일랜드 라이프 (2D)

요청대로 앞으로도 계속 고도화할 수 있도록, 이번 커밋은 **전반 시스템 1단계 기반 강화**에 집중했습니다.

## 이번 단계에서 바뀐 것 (전반 고도화 1단계)

- **전역 진행 기반 추가**
  - 날짜/계절/날씨 시스템
  - 진행 로그(사이드 HUD)
  - 로컬 저장/불러오기(LocalStorage)
- **경제/목표 루프 확장**
  - 단일 퀘스트가 아닌 단계형 퀘스트 체인(3단계)
  - 레벨업 루프 정비
- **재료 활용처 확대**
  - 제작: 가구/다리/미끼/씨앗
  - 상점: 집 구매/업그레이드/씨앗 패키지
- **농사 시스템 도입**
  - 농지(FARM) 구역
  - 씨앗 심기(`R`) → 성장 단계 → 수확
- **NPC 운영 개선**
  - 시간대별 기본 스케줄(낚시/농사/산책/사교)
  - 기존 말풍선 상호작용 유지
- **UI 확장**
  - 마을보드 버튼 및 모달
  - 진행 로그 패널 추가

## 기존 핵심 시스템 유지

- 다리 설치 후 다리 위 낚시
- 타이밍 낚시 미니게임 (입질/릴링)
- 집 구매/업그레이드/입장/인테리어 배치
- 말풍선 대화 + 상호작용 중 정지/바라보기

## 조작

- 이동: `WASD` / 방향키
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 농사 심기/수확: `R`
- 대화 선택: `1` / `2`, 종료: `Esc`

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
