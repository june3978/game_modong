# 힐링 포레스트: 아일랜드 라이프

현재 버전은 **v1.8** 입니다.

## 이번 업데이트(v1.8) - 3D 조작/대화/방향 동기화 개선

- **3D 방향키 조작 버그 수정**
  - 카메라 yaw 기준 전/후/좌/우 벡터를 재계산해, 시점을 돌렸을 때도 입력 방향이 직관적으로 동작
- **3D 캐릭터 바라보기 동기화**
  - 플레이어/NPC 3D 모델 회전을 `facing` 값과 일치하도록 반영
  - 대화 시작 시 서로 바라보는 방향이 3D에서도 자연스럽게 유지
- **3D 대화창 오버레이 추가**
  - 2D 캔버스 전용으로만 보이던 대화 선택 UI를 3D 모드에서도 표시
  - 선택지/종료 안내(1,2,Esc) 포함
- **NPC 이동 벡터 추적 보강**
  - NPC의 `vx/vy` 값을 업데이트해 걷기 모션 판정 정합성 개선

## 조작

- 이동: `WASD` / 방향키 (3D 모드에서는 카메라 시점 기준)
- 달리기: `Shift`
- 상호작용: `E`
- 낚시/타이밍: `Space`
- 인테리어 배치: `F`
- 가구 편집: `I/J/K/L`, `T`, `Tab`
- 농사 심기/수확: `R`
- 곤충 채집: `G`
- 대화 선택: `1` / `2`, 종료: `Esc`
- 렌더 전환: 상단 `3D뷰` 버튼
- 3D 카메라: `Q/C` 회전, `Z/X` 줌

## 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속.
